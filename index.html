Assembly Line Textual Simulation Application

Programmed in C++ for an academic assignment, this was a modelling of an assembly line simulator utilizing multiple classes to resemble an order filling process. Multiple customers would have products that would be checked and filled based on the determined queue, and many intermediate general programming topics such as standard library containers, iterators, and algorithms, inheritance, and lambda functions were utilized to achieve this functionality.

Sample Registration Form

A comprehensive registration form sample created for an academic assignment. This form utilizes many features of HTML forms and validation on client-side, and sends the information as JSON data to httpbin.org, a simple HTTP request & response service.

Experience Calculator - Brave Frontier

One of my first public web projects that still sees some usage today. This is an experience calculator for a mobile game, where the user inputs their current and desired scenarios, and determines how many times they will have to complete a certain stage in order to reach their goals. The website utilizes standard HTML/CSS/Javascript, with many preset fields to minimize manual data insertion and potential errors.
